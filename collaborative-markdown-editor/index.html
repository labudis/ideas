Product Launch Plan

Target launch date: March 28, 2026

Overview

This document outlines the launch plan for the collaborative markdown editor. The editor supports real-time collaboration, version history, and inline commenting.

Key Features

  • Real-time cursor presence and selections
  • Inline threaded comments on any line
  • Version history with diff view
  • Split-pane editing (source + preview)
  • File tree with multi-doc workspaces
  • Markdown toolbar with shortcuts
